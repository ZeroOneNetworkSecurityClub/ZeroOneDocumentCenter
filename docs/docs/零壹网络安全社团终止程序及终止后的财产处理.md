# 零壹网络安全社团终止程序及终止后的财产处理

#### 一、终止条件

*   **法定终止情形：**
    
    *   社团宗旨已完成或无法实现；
        
    *   成员人数持续低于法定最低人数（如不足10人）且无法恢复；
        
    *   违反《网络安全法》《社会团体登记管理条例》等法规被主管部门责令终止； 其他经会员大会决议通过的情形。
        
    *   因分立、合并需终止；
        
*   **特殊终止情形：**
    
    *   连续两年未开展网络安全相关活动；
        
    *   因技术安全事件导致重大声誉或法律风险，经评估无法修复。
        

#### 二、终止程序

1.  **终止动议提出**
    

*   由执行委员会或10人以上成员联名提出终止动议，提交书面申请至社团理事会。
    
*   动议需明确终止原因、资产及债务概况、后续处理预案，并附相关证明材料（如成员名单、活动记录、财务报表）。
    

**2. 会员大会表决**

*   召开全体成员大会或代表大会，提前7日公示动议内容、清算组成员名单及初步方案。
    
*   表决采用无记名投票或线上电子投票，需经出席成员2/3以上同意且同意人数超过全体成员半数。
    
*   表决结果需形成会议纪要，由出席成员签字确认，并报业务主管单位（如学校团委）备案。
    

**3. 清算组成立与职责**

*   清算组由业务主管单位代表、社团负责人、财务人员、会员代表及外部审计机构组成，人数不少于3人。
    
*   清算组职责包括：
    
    *   清理社团资产、债权债务，编制资产负债表及财产清单；
        
    
    *   处理未了结事务，如未完成的项目、合同终止协商；
        
    *   制定清算方案，包括资产处置、债务清偿、剩余财产分配计划；
        
    *   定期向会员大会及业务主管单位报告清算进展。
        

**4. 清算执行**

资产清查：全面盘点固定资产（如服务器、电脑）、流动资产（如现金、应收账款）、无形资产（如软件许可证、知识产权），聘请专业机构评估价值。 债务清偿：按顺序支付清算费用（如审计费、法律顾问费）、职工工资及社保、税款、其他债务。 数据安全处理： 敏感数据（如成员个人信息、技术活动记录）进行脱敏处理或安全销毁，确保无法恢复； 存储介质（如硬盘、U盘）由专业机构进行物理销毁或数据擦除，保留销毁记录及视频证据； 信息系统（如社团官网、数据库）终止前进行安全审计，确保无残留数据或后门程序。

**5. 注销登记与公告**

*   清算结束后15日内，向社团登记管理机关   提交以下材料：
    
    *   法定代表人签署的注销登记申请书；
        
    *   业务主管单位审查同意文件；
        
    *   清算报告书（含资产处置、债务清偿、剩余财产分配明细）；
        
    *   社团登记证书、印章、财务凭证。
        
    *   登记机关核准后，发布注销公告，收回社团登记证书，终止社团法人资格。
        

#### 三、终止后财产处理

**剩余财产分配原则：**

*   优先用于支付清算费用及债务清偿；
    
*   剩余财产在业务主管单位及登记管理机关监督下，用于与社团宗旨相关的网络安全公益事业，如捐赠给
    
*   同类社团、支持网络安全教育项目；
    
*   禁止向成员或关联方分配剩余财产。
    

**特殊资产处理：**

1.  技术资产（如服务器、软件许可证）可转让给其他合法组织或进行安全销毁，确保无敏感信息泄露；
    
2.  知识产权（如专利、著作权）可转让或捐赠，需签订正式转让协议并办理登记手续；
    
3.  资金账户余额转入指定公益账户，用于网络安全公益项目，保留资金流向记录备查。
    

#### 四、数据安全与保密

1.  终止过程中，所有涉及敏感信息的操作需由具备资质的第三方机构执行，并签订保密协议；
    
2.  成员个人信息、技术活动记录等敏感数据需在终止后30日内完成脱敏或销毁，确保符合《个人信息保护法》《数据安全法》要求；
    
3.  终止后，社团官网、社交媒体账号等需关闭或转移，避免信息泄露风险。
    

#### 五、监督与争议解决

1.  成立终止监督小组，由业务主管单位、指导教师、会员代表及法律顾问组成，监督终止过程合法合规；
    
2.  成员对终止程序或财产处理有异议的，可向监督小组提出书面申诉，监督小组应在15日内调查并反馈处理意见；
    
3.  争议无法解决的，可向社团登记管理机关或司法机关申请仲裁或诉讼。
    

#### 六、附则

1.  本方案经会员大会表决通过后生效，原社团章程中相关条款同步废止；
    
2.  解释权归零壹网络安全社团执行委员会所有；
    
3.  终止程序及财产处理需符合国家法律法规及学校社团管理规定，确保公开、透明、合法。
